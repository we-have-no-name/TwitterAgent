# TwitterAgent
simple twitter agent to help us in making the Data set 

you nead to make a simple config.json file with this structure
```json

{

	"consumer_key": "",
	"consumer_secret": "",

	"access_token":"",
	"access_token_secret":""
}


```


and pass it to the TwitterAgent object 



```python

agent = TwitterAgent("config.json")

```
